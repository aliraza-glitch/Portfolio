Quadratics

Key Formulas
x = (−b ± √(b²−4ac)) / 2a
Quadratic Formula
Solves ax² + bx + c = 0 for any values of a, b, c
Δ = b² − 4ac
Discriminant
Δ > 0: two real roots · Δ = 0: one repeated root · Δ < 0: no real roots
y = a(x − h)² + k
Vertex (Completed Square) Form
Vertex at (h, k). Axis of symmetry: x = h
x = −b / 2a
Axis of Symmetry
x-coordinate of the turning point
Sum of roots = −b/a
Vieta's Formulas
Product of roots = c/a. For ax²+bx+c=0 with roots α, β
(x + y)² = x² + 2xy + y²
Perfect Square Expansion
(x − y)² = x² − 2xy + y²
x² − y² = (x+y)(x−y)
Difference of Two Squares
Useful for factorisation
Interactive — Quadratic Graph Explorer

Completing the Square — Method

Step 1

Write ax² + bx + c. Factor out a from the x terms: a(x² + (b/a)x) + c

Step 2

Add and subtract (b/2a)²: a[(x + b/2a)² − (b/2a)²] + c

Step 3

Expand and simplify to get a(x − h)² + k where h = −b/2a, k = c − b²/4a

Why it works

Vertex form immediately tells you the turning point (h, k) and whether it's a min or max

Polynomials & Factor Theorem

Key Theorems
If f(a) = 0, then (x − a) is a factor of f(x)
Factor Theorem
Test potential factors by substitution. If result is 0, it's a factor.
f(a) = remainder when f(x) is divided by (x − a)
Remainder Theorem
No need to do full division — just substitute x = a into f(x)
f(x) = (x−a) · Q(x) + R
Polynomial Division Result
Q(x) = quotient, R = remainder
x³ + y³ = (x+y)(x²−xy+y²)
Sum of Cubes
x³ − y³ = (x−y)(x²+xy+y²)

Surds & Indices

Surd Rules
√(ab) = √a · √b
Product Rule
√(a/b) = √a / √b
Quotient Rule
(√a)² = a
Square of surd
a / (p + q√b) → multiply by (p − q√b)/(p − q√b)
Rationalising the Denominator
Conjugate pair: (p+q√b)(p−q√b) = p²−q²b
Index Laws
aᵐ · aⁿ = aᵐ⁺ⁿ
Multiplication
aᵐ / aⁿ = aᵐ⁻ⁿ
Division
(aᵐ)ⁿ = aᵐⁿ
Power of a Power
a⁰ = 1
Zero Power
For any a ≠ 0
a⁻ⁿ = 1 / aⁿ
Negative Index
a^(1/n) = ⁿ√a
Fractional Index (root)
a^(m/n) = ⁿ√(aᵐ) = (ⁿ√a)ᵐ
Fractional Index (general)

Logarithms & Exponentials

Log Laws
y = aˣ ⟺ x = log_a(y)
Definition of Logarithm
log_a(y) asks: "to what power must a be raised to get y?"
log(AB) = log A + log B
Product Rule
log(A/B) = log A − log B
Quotient Rule
log(Aⁿ) = n log A
Power Rule
log_a(a) = 1
Log of its own base
log_a(1) = 0
Log of 1
log_b(x) = log(x) / log(b)
Change of Base
Convert between any bases using this formula
ln(x) = log_e(x), e ≈ 2.718
Natural Logarithm
ln(eˣ) = x and e^(ln x) = x

Binomial Expansion

Formulas & Pascal's Triangle
(a + b)ⁿ = Σ C(n,r) · aⁿ⁻ʳ · bʳ (r = 0 to n)
Binomial Theorem
C(n,r) = n! / (r!(n−r)!) = ⁿCᵣ — the binomial coefficient
C(n,r) = n! / (r!(n−r)!)
Binomial Coefficient
Also written ⁿCᵣ or (n choose r)
T(r+1) = C(n,r) · aⁿ⁻ʳ · bʳ
(r+1)th Term (General Term)
r starts at 0; use this to find a specific term
(1+x)ⁿ ≈ 1 + nx + n(n−1)x²/2! + ...
Binomial Approximation (|x| < 1)
Valid for any n when |x| < 1

Inequalities

Rules & Methods
Flip the inequality when multiplying or dividing by a NEGATIVE number
Critical Rule
e.g. −2x > 4 → x < −2
ax² + bx + c > 0 or < 0
Quadratic Inequality
Sketch the parabola and read off where it's above/below x-axis. Always find roots first.
|x| < a ⟺ −a < x < a
Modulus Inequality
|x| > a ⟺ x < −a or x > a
Linear: isolate x, keeping direction
Linear Inequality
Same as solving an equation, except flip sign when ÷ or × by negative
