Statistics & Probability

Measures of central tendency, spread, probability, permutations, combinations, and distributions.

01

Measures of Central Tendency & Spread

Averages
x̄ = (Σx) / n
Mean
Sum of all values ÷ number of values. Σfx / Σf for grouped data.
Median = middle value when ordered
Median
For n values: position (n+1)/2. For even n: average of two middle values.
Mode = most frequent value
Mode
Can be more than one mode (bimodal, multimodal)
Measures of Spread
σ² = Σ(x − x̄)² / n = Σx²/n − x̄²
Variance
Second formula (Σx²/n − x̄²) is easier to compute. Units are squared.
σ = √variance
Standard Deviation
Same units as the data. Measure of spread around the mean.
Range = max − min
Range
IQR = Q3 − Q1
Interquartile Range
Q1 = lower quartile (25th percentile), Q3 = upper quartile (75th percentile)
s² = Σ(x−x̄)² / (n−1)
Sample Variance (unbiased)
Use n−1 (Bessel's correction) for sample estimating population
02

Probability

Core Rules
P(A) = favourable outcomes / total outcomes
Basic Probability
0 ≤ P(A) ≤ 1 always. P(impossible) = 0. P(certain) = 1.
P(A') = 1 − P(A)
Complement
P(A) + P(A') = 1
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
Addition Rule
Avoid double-counting the intersection
P(A ∩ B) = 0 → mutually exclusive
Mutually Exclusive Events
Can't both happen. P(A ∪ B) = P(A) + P(B)
P(A ∩ B) = P(A)·P(B) → independent
Independent Events
Occurrence of A doesn't affect B
P(A|B) = P(A ∩ B) / P(B)
Conditional Probability
"Probability of A given B has occurred"
P(A ∩ B) = P(A|B) · P(B)
Multiplication Rule
03

Permutations & Combinations

Counting Formulas
n! = n × (n−1) × ... × 2 × 1
Factorial
0! = 1 by definition
ⁿPᵣ = n! / (n−r)!
Permutations
Ordered arrangements of r items from n. ORDER MATTERS.
ⁿCᵣ = n! / (r!(n−r)!)
Combinations
Selections of r items from n. ORDER DOESN'T MATTER.
ⁿCᵣ = ⁿCₙ₋ᵣ
Symmetry of Combinations
Total arrangements = n! / (n₁! n₂! ...)
Arrangements with Repeats
Divide by factorial of each repeated group
04

Probability Distributions

Discrete Random Variables
Σ P(X = x) = 1 for all x
Probability Distribution Condition
All probabilities must sum to 1
E(X) = Σ x · P(X = x)
Expected Value (Mean)
The theoretical long-run average
Var(X) = E(X²) − [E(X)]²
Variance
E(X²) = Σ x² · P(X=x)
E(aX + b) = aE(X) + b
Linear transformation of E(X)
Var(aX + b) = a²Var(X)
Linear transformation of Var(X)
Adding a constant b does NOT affect variance
Binomial Distribution B(n, p)
P(X = r) = ⁿCᵣ · pʳ · (1−p)ⁿ⁻ʳ
Binomial Probability
n = number of trials, p = probability of success, r = number of successes
E(X) = np
Binomial Mean
Var(X) = np(1−p) = npq
Binomial Variance
q = 1 − p = probability of failure
Conditions: fixed n, each trial independent, constant p, only two outcomes (success/failure)
When to use Binomial
Normal Distribution N(μ, σ²)
X ~ N(μ, σ²)
Normal Distribution Notation
μ = mean, σ² = variance, σ = standard deviation. Bell-shaped, symmetric about μ.
Z = (X − μ) / σ
Standardisation (Z-score)
Z ~ N(0,1) — standard normal. Use Z-tables to find probabilities.
P(X < a) = P(Z < (a−μ)/σ)
Finding Probabilities
P(Z < −z) = P(Z > z) = 1 − P(Z < z)
Symmetry of Normal
68% within 1σ · 95% within 2σ · 99.7% within 3σ
Empirical Rule (68-95-99.7)
These are approximate. Use Z-tables for exact values.
