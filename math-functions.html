Domain, range, composite and inverse functions, transformations, and interactive graph explorer.

01

Functions — Core Concepts

Definitions
A function maps each input x in the domain to exactly one output y in the range
Function Definition
Use vertical line test: if any vertical line crosses graph more than once, it's not a function
Domain — set of all valid inputs (x values)
Domain
Watch out for: division by zero, even roots of negatives, log of non-positive numbers
Range — set of all possible outputs (y values)
Range
Find by considering shape of graph and any restrictions
fg(x) = f(g(x))
Composite Function
Apply g first, then apply f to the result. Domain of fg = domain of g restricted so g(x) is in domain of f
ff⁻¹(x) = f⁻¹f(x) = x
Inverse Function
Swap x and y, then rearrange to find f⁻¹(x). Domain of f⁻¹ = range of f.
Graph of f⁻¹(x) is reflection of f(x) in the line y = x
Inverse — graphical meaning
f must be one-to-one (bijective) to have an inverse. If not, restrict the domain first.
02

Graph Transformations

Transformation Rules
f(x) + a
Vertical Translation
Shift UP by a (or down if a < 0). Affects y-values.
f(x + a)
Horizontal Translation
Shift LEFT by a (counterintuitive). Affects x-values.
a·f(x)
Vertical Stretch
Scale factor a in y-direction. Multiply all y-values by a.
f(ax)
Horizontal Stretch
Scale factor 1/a in x-direction. Divide all x-values by a.
−f(x)
Reflection in x-axis
Negate all y-values
f(−x)
Reflection in y-axis
Negate all x-values
|f(x)|
Modulus of function
Reflect any part below x-axis to above it
f(|x|)
Modulus of input
Reflect right half of graph to replace left half

Types of Functions & Graphs

Common Function Types

Linear: y = mx + c

Straight line. Gradient m, y-intercept c. Domain: ℝ, Range: ℝ

Quadratic: y = ax² + bx + c

Parabola. a > 0: U-shape (min). a < 0: ∩-shape (max). Axis of symmetry x = −b/2a

Cubic: y = ax³ + ...

S-shaped curve. Has up to 2 turning points, up to 3 real roots.

Reciprocal: y = k/x

Hyperbola. Asymptotes at x=0 and y=0. Domain: x≠0, Range: y≠0

Exponential: y = aˣ

Always positive. y-intercept at (0,1). Asymptote y=0. Domain: ℝ, Range: y>0

Logarithm: y = log x

x-intercept at (1,0). Asymptote x=0. Domain: x>0, Range: ℝ
