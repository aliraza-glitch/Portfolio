01

Differentiation

First Principles
f'(x) = lim[h→0] (f(x+h) − f(x)) / h
Definition from First Principles
The gradient of the tangent at a point — the fundamental definition of derivative
Standard Derivatives
d/dx [xⁿ] = nxⁿ⁻¹
Power Rule
Bring down the power, reduce power by 1. Works for any n.
d/dx [c] = 0
Constant
d/dx [eˣ] = eˣ
Exponential
d/dx [ln x] = 1/x
Natural Log
d/dx [sin x] = cos x
Sine
d/dx [cos x] = −sin x
Cosine
d/dx [tan x] = sec²x
Tangent
d/dx [aˣ] = aˣ ln a
General Exponential
Differentiation Rules
d/dx [uv] = u'v + uv'
Product Rule
Use when two functions are multiplied together
d/dx [u/v] = (u'v − uv') / v²
Quotient Rule
Use when one function is divided by another
dy/dx = dy/du × du/dx
Chain Rule
For composite functions. "Differentiate outside × inside derivative"
d/dx [f(ax+b)] = a·f'(ax+b)
Chain Rule (linear inside)
Quick version for linear inner functions
d/dx [cf(x)] = c·f'(x)
Scalar Multiple
d/dx [f ± g] = f' ± g'
Sum/Difference Rule
Applications of Differentiation
dy/dx = 0 → stationary point. d²y/dx² > 0: min · < 0: max · = 0: check further
Stationary Points
Second derivative test. If d²y/dx² = 0, use sign change of dy/dx to determine nature.
Gradient of tangent at x=a: m = f'(a)
Tangent Line
Equation: y − f(a) = f'(a)(x − a)
Gradient of normal = −1 / f'(a)
Normal Line
Normal is perpendicular to tangent
Rate of change = dy/dx at that instant
Rates of Change
v = ds/dt · a = dv/dt = d²s/dt²

02

Integration

Standard Integrals
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
Power Rule (n ≠ −1)
Add 1 to power, divide by new power, add constant C
∫ 1/x dx = ln|x| + C
Reciprocal
∫ eˣ dx = eˣ + C
Exponential
∫ sin x dx = −cos x + C
Sine
∫ cos x dx = sin x + C
Cosine
∫ sec²x dx = tan x + C
Secant squared
∫ k dx = kx + C
Constant
∫ (ax+b)ⁿ dx = (ax+b)ⁿ⁺¹ / (a(n+1)) + C
Linear composite
Definite Integrals & Area
∫[a to b] f(x) dx = F(b) − F(a) where F'(x) = f(x)
Fundamental Theorem of Calculus
F(x) is the antiderivative. This gives the signed area under f(x) between x=a and x=b.
Area = |∫[a to b] f(x) dx|
Area under curve
If curve goes below x-axis, integrate separately and add magnitudes
Area between curves = ∫[a to b] (f(x) − g(x)) dx
Area between two curves
f(x) is the upper curve, g(x) is lower. Find intersections first (set f = g).
∫ by substitution: let u = g(x), du = g'(x)dx
Integration by Substitution
Change limits too if definite. Replace all x terms with u.
∫ u dv = uv − ∫ v du
Integration by Parts
LIATE rule: choose u as Logarithm, Inverse, Algebraic, Trig, Exponential (in that preference)
